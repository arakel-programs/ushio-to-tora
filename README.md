# 2D RPG System Prototype (Godot)

A modular 2D RPG gameplay system built with Godot Engine.

## Overview

This project demonstrates a structured RPG architecture including:

- Player movement system
- Encounter trigger logic
- Battle scene transitions
- Dialogue box system
- Fade transition effects
- Scene-based modular design

## Architecture

The project follows a modular scene structure:

- `World.tscn` – Main exploration scene
- `Battle.tscn` – Combat scene
- `DialogueBox.tscn` – Dialogue system
- `Fade.tscn` – Scene transition effects
- `player.gd` – Player movement logic
- `enemy.gd` – Enemy behavior
- `Encounter.gd` – Encounter trigger system

## Features

- Event-driven battle initiation
- State-based gameplay flow
- Clean scene separation
- Reusable systems
- Expandable architecture

## Tech Stack

- Godot Engine
- GDScript
- Scene-based architecture
